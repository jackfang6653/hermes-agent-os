# Hermes OS 统一搜索

## 搜索源
- GitHub Repos: gh search repos
- GitHub Code: gh search code
- Web: Firecrawl
- 社区: gh search issues
- 本地知识库: .hermes/knowledge/

## 搜索流程
1. 明确搜索意图和关键词
2. 选择最合适的搜索源
3. 构造搜索查询
4. 执行并过滤结果
5. 保存到知识库
