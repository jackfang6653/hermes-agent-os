# 搜索机制 SOP

## 1. GitHub 搜索 (通过 gh CLI)
- 搜索仓库: gh search repos
- 搜索代码: gh search code
- 搜索 issue: gh search issues
- 搜索 PR: gh search prs

## 2. Web 搜索 (Firecrawl)
- 技术文档搜索
- API 参考查询
- 最佳实践搜索
- 错误解决方案查找

## 3. 社区搜索
- Stack Overflow
- GitHub Discussions
- Reddit
- Discord / Telegram 群组

## 4. 本地知识库 (RAG)
- \.hermes/knowledge/\ 目录
- 技术调研报告
- 架构决策记录
- 项目 FAQ

## 5. Research Agent 流程
1. 明确搜索目标和关键词
2. 选择搜索源 (GitHub/Web/社区)
3. 执行搜索并过滤结果
4. 提取关键信息
5. 保存到知识库
6. 向 CEO 报告发现
